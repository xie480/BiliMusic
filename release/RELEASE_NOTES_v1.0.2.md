# 🎵 Nyami v1.0.2 发布说明

> 本次更新为紧急修复版本，重点解决了后台播放时应用容易被系统回收的问题，大幅提升了后台播放的稳定性。

---

## 🐛 问题修复

- **后台播放**：修复了应用在后台播放时，JavaScript 线程容易被系统杀死导致播放中断的问题。现在您可以更放心地在后台享受音乐了。

---

## 📦 安装与更新

- **系统要求**：Android 7.0+  
- **安装方式**：下载 APK 文件直接覆盖安装即可
