# 🎵 Nyami v1.0.0 发布说明

> 首个正式版本，代号「Nyami」。  
> Nyami 是一款基于 React Native 的第三方 B 站音乐播放器，专注于提供纯净、沉浸式的音乐体验。

---

## ✨ 核心功能亮点

### 🔐 B 站账号与数据同步
- **扫码登录**：通过二维码一键登录 B 站账号，安全便捷
- **收藏夹同步**：完整同步你的 B 站收藏夹，支持**游标增量同步**与**全量索引机制**，确保数据与 B 站实时一致
- **智能限流**：内置 **AIMD 自适应限流**策略，避免触发 B 站风控，保障同步稳定
- **过滤与排序**：可按收藏夹名称过滤，保持与 B 站相同的展示顺序

### 🎧 沉浸式音乐播放
- **多模式播放**：支持列表循环、单曲循环、随机播放、顺序播放等多种模式
- **队列管理**：灵活的播放队列，支持动态增删曲目，自动保持播放状态
- **分P视频支持**：自动识别分P视频，可按顺序切换多段音频
- **智能缓存**：多 CDN 并发测速，自动选择最优线路，提升加载速度
- **离线缓存**：支持音乐文件本地缓存，无网络也能畅听

### 🎨 玻璃拟态（Glassmorphism）主题
- **磨砂玻璃风格**：全局 UI 采用玻璃拟态设计，毛玻璃模糊背景，视觉通透且现代
- **动态背景**：根据当前播放内容自适应背景，沉浸感更强
- **主题切换**：支持亮色/暗色模式切换，并预留主题扩展能力

### 🎛️ 专业级音频均衡器
- **10 段图形均衡器（EQ）**：支持 ±12dB 增益调节，实时生效
- **参数式均衡器（PEQ）**：支持频率、增益、Q 值精细调节，满足高阶调音需求
- **预设管理**：内置多种预设音效，并支持自定义保存
- **实时频谱可视化**：播放时显示实时音频频谱，随音乐律动

### 🔍 搜索与浏览
- **站内搜索**：支持搜索 B 站音乐视频，快速发现新内容
- **索引浏览**：提供全局索引查询界面，快速定位已同步的收藏夹与视频
- **多视图切换**：支持文件列表、可见文件夹等多种浏览方式

### 🛠️ 其他特性
- **日志系统**：内置分级日志记录与导出功能，便于问题反馈与诊断
- **高性能渲染**：多层渲染优化，避免卡顿与重复渲染
- **原生状态栏适配**：完美兼容 Android 原生状态栏，沉浸式全屏体验
- **应用图标与品牌**：全新 Nyami 标识，焕然一新的品牌形象

---

## 🚀 技术架构升级

- **存储层重构**：引入 **WatermelonDB** 高性能本地数据库，解决大规模数据场景下的内存溢出问题，查询更高效
- **限流与容灾**：从简单的延迟策略升级为 **AIMD 自适应限流 + 降级策略**，保障同步完成度
- **底层处理优化**：音频播放引擎、原生模块（DSP、FFT）全面优化，播放更稳定，延迟更低

---

## 📦 安装与更新

- **系统要求**：Android 7.0+  
- **安装方式**：下载 APK 文件直接安装（当前版本暂未上架应用商店）  
- **已知问题**：首次同步可能耗时较长（取决于收藏夹数量），建议在稳定网络环境下进行
