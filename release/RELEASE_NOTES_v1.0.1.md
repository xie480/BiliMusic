# 🎵 Nyami v1.0.1 发布说明

> 本次更新为日常维护版本，主要集中在问题修复与体验优化，进一步提升了应用的稳定性与流畅度。

---

## ✨ 优化与改进

- **界面与视觉**：更新了磨砂玻璃（明亮）主题的默认背景，优化了文本换行显示效果
- **交互体验**：优化了设置界面的 Toast 提示，以及日志导出功能的提示信息
- **性能提升**：优化了整体渲染性能，并进一步压缩了 APK 包体积
- **功能增强**：新增了部分设置项，丰富了 EQ 预设配置
- **文档更新**：README.md 新增了相关注意事项

## 🐛 问题修复

- **界面适配**：修复了播放界面的机型适配问题，解决了播放界面重复渲染的 Bug
- **播放控制**：修复了自动播放和切曲时可能出现的暂停问题，以及视频列表（VideosScreen）播放音频队列的并发问题
- **代码混淆**：修复了 ProGuard/R8 代码混淆导致的部分功能异常问题
- **其他修复**：修复了搜索组件的非响应式问题，以及一些零碎的类型错误

## ⚙️ 构建与工程

- 配置了应用签名密码
- 优化了项目代码规范

---

## 📦 安装与更新

- **系统要求**：Android 7.0+  
- **安装方式**：下载 APK 文件直接覆盖安装即可
